How to add your photos:
1. Create an assets/ folder next to index.html in your repo.
2. Copy your photos there — e.g. assets/conf1.jpg, assets/conf2.jpg …
3. In this file, replace each placeholder <div class="g-item"> with the <img> version shown in the comments above.
4. Update the caption text with the event name and location.
